# UCS654
## Probability Distribution and PDF Fitting

### Objective
To generate synthetic Gaussian data and fit a Probability Density Function (PDF) to verify distribution assumptions used in Machine Learning.

---

## Tools Used
- Python
- NumPy
- Matplotlib
- SciPy
- Kaggle Notebook

---

## Steps Performed
1. Generated synthetic data using Gaussian distribution.
2. Calculated statistical parameters (Mean, Standard Deviation).
3. Visualized empirical distribution using histogram.
4. Fitted Gaussian PDF using statistical parameters.
5. Compared empirical and theoretical distribution.

---

## Results

- Number of Samples: 1000  
- Mean ≈ 50  
- Standard Deviation ≈ 10  
- Skewness ≈ 0  
- Kurtosis ≈ 0  
- Distribution: Approximately Gaussian  

---

## Conclusion

The histogram closely matches the fitted Gaussian PDF.  
The skewness and kurtosis values confirm that the data approximately follows a normal distribution.

This validates the assumption of normally distributed input features in Machine Learning.
